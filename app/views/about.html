This is CS50 and this is my final project!

A simple web application written in AngularJs (mostly) and jQuery. I used also MaterializeCSS and AnimateCSS for the styling. The goal is to get the PageSpeed API results, formatting and make a pretty print of it, allowing the user to analyze all the specs from every platform (including Smartophone and Tablet).

I have written a back-end service in NodeJs using also the Googleapis NodeJs Client to get the results.

Both projects, ng-PageSpeed and express-pgspeedapi are available on GitHub.
